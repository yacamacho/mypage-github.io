En HTML5 no es necesario proporcionar valore para los atributos, como los cotroles, reproduccion automaticas y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo esta presente, activa esa opcion. Si se omite el atributo, la opcion esta desactivada. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrará lo que esté entre la <video> de apertura y laas etiquetas de </video> de cierre.

preload: este atributo le indica al navegador que hace cuando se carga la pagina. Puede tener tres valores:
none: el navegador no debe cargar el video hasta que el usuario presiona play.
auto: el navegador debe descargar el video cuando se carga la pagina.
metadata: el navegador debe simplemente recopilar la informacion como el tamaño, el primer fotograma, la lista de pistas la dración.
