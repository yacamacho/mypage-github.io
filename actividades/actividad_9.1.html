Cada elemento HTML puede llevar id. se utiliza para identificar de forma unica ese elemento de otros elementos de la pagina. Su valor debe comenzar con una letra o un guion bajo (no un numero o cualquier otro caracter). Es importante que no hay dos elementos en la misma pagina tiene el mismo valor para sus atributos ID (de lo contrario, el valor ya no es unico).

por ejemplo, es posible que desee asignar un parrafodentro de la pagina (quizas un parrafo que contiene una cotizacion de extraccion) un estilo diferente que todos los demas parrafos. En el ejemplo, el parafo con el atributo ID cuyo valor es pullquote se hace en mayuscula usando CSS.
